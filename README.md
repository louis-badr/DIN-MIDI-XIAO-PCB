# DIN-MIDI-XIAO-PCB
DIN MIDI PCB for Seeed Studio XIAO format boards - designed for CNC milling

Pinout:
- Pin 1 -> GND
- Pin 2 -> MIDI- / 5
- Pin 3 -> MIDI+ / 4
- Pin 4 -> +5V

Pin 1 and 4 are for powering the XIAO board, only connect them when USB is not plugged in.

Sample code is in the `firmware` folder.
